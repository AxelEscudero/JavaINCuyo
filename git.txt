¿Qué es git y para qué sirve?
Git es un sistema de control de versiones distribuido, lo que significa que un clon local del proyecto es un repositorio de control de versiones completo. Estos repositorios locales plenamente funcionales permiten trabajar sin conexión o de forma remota con facilidad.
¿Para qué sirve, y cómo crees que va a ayudarte a usarlo en programación?
En términos generales, GitHub es un servicio en la nube que permite a los desarrolladores y desarrolladoras almacenar y administrar su código, así como a llevar el registro y control de cualquier cambio que se realice sobre este código.
